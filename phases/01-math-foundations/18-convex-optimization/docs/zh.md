# 凸优化 (Convex Optimization)

> 凸问题只有一个谷底。神经网络有数百万个。知道区别很重要。

**类型：** 构建 (Build)

## 学习目标

- 使用定义、二阶导数和 Hessian 准则检验函数是否为凸函数
- 实现牛顿法 (Newton's Method)，并将其二次收敛速度与梯度下降对比
- 使用拉格朗日乘数法求解约束优化问题，并解释 KKT 条件

## 问题

第 08 课教你梯度下降、动量和 Adam。这些优化器在任何曲面上都能走下坡路，但它们没有任何保证。非凸地形上的梯度下降可能落入糟糕的局部最小值、卡在鞍点上或永远振荡。你仍然使用它，因为神经网络是非凸的，没有替代方案。

但机器学习中的许多问题是凸的。线性回归、逻辑回归、SVM、LASSO、岭回归——对于这些，存在更强的东西：有数学保证的优化。凸问题恰好有一个谷底。任何走下坡路的算法都会到达全局最小值——不需要重启动，不需要学习率调度，不需要祈祷。

理解凸性做三件事。第一，它告诉你什么时候你的问题是容易的（凸）或困难的（非凸）。第二，它为凸问题提供更快的工具，如牛顿法。第三，它解释了贯穿 ML 的概念：正则化作为约束、SVM 中的对偶性，以及为什么深度学习尽管违反了凸性给出的所有优良属性却仍然有效。

## 概念

### 凸集

如果集合 S 中的任意两点之间的线段也完全位于 S 中，则 S 是凸集。

| 凸集 | 非凸 |
|------|------|
| **矩形**：内部的任意两点可以用一条完全在内部的线段连接 | **星形/新月形**：两个内部点之间的线可能穿过集合外部 |
| **三角形**：同样性质对所有内部点成立 | **甜甜圈/环**：中间的洞意味着某些线段离开集合 |
| 任意两点之间的线段保持在集合内 | 某些点对之间的线段离开集合 |

形式化检验：对于 S 中的任意点 x、y 和 [0, 1] 中的任意 t，点 tx + (1-t)y 也在 S 中。

凸集的例子：
- 一条线、一个平面、整个 R^n
- 球（圆、球体、超球体）
- 半空间：{x : a^T x <= b}
- 任意数量凸集的交集

非凸集的例子：
- 甜甜圈（环）
- 两个分离圆的并集
- 任何有"凹陷"或"洞"的集合

### 凸函数

如果函数 f 的定义域是凸集，且对定义域中任意两点 x、y 和 [0, 1] 中的任意 t：

```
f(tx + (1-t)y) <= t*f(x) + (1-t)*f(y)
```

则 f 是凸函数。

几何上：图上任意两点之间的线段位于图的上方或在图上。

| 性质 | 凸函数 | 非凸函数 |
|------|--------|---------|
| **线段检验** | 图上任意两点之间的线位于曲线**上方** | 图上某些点之间的线落在曲线**下方** |
| **形状** | 单一碗状/谷状，向上弯曲 | 多个峰和谷，曲率混合 |
| **局部最小值** | 每个局部最小值是全局最小值 | 可能存在多个不同高度的局部最小值 |

常见凸函数：
- f(x) = x^2（抛物线）
- f(x) = |x|（绝对值）
- f(x) = e^x（指数）
- f(x) = max(0, x)（ReLU，尽管是分段线性的）
- f(x) = -log(x)，x > 0（负对数）
- 任何线性函数 f(x) = a^T x + b（既是凸的也是凹的）

### 检验凸性

三种实际检验方法，从最简单到最严格。

**检验 1：二阶导数检验（一维）。** 如果对所有 x 有 f''(x) >= 0，则 f 是凸的。

- f(x) = x^2：f''(x) = 2 >= 0。凸。
- f(x) = x^3：f''(x) = 6x。x < 0 时为负。非凸。
- f(x) = e^x：f''(x) = e^x > 0。凸。

**检验 2：Hessian 检验（多变量）。** 如果 Hessian 矩阵 H(x) 对所有 x 是半正定的，则 f 是凸的。Hessian 是二阶偏导数矩阵。

**检验 3：定义检验。** 直接检查不等式 f(tx + (1-t)y) <= t*f(x) + (1-t)*f(y)。适用于导数难以计算的函数。

### 为什么凸性重要

凸优化的核心定理：

**对凸函数，每个局部最小值都是全局最小值。**

这意味着梯度下降不会被困住。任何下坡路径都通向相同的答案，算法保证收敛到最优解。

后果：
- 不需要随机重启动
- 不需要复杂的学习率调度
- 收敛性证明是可能的（速率取决于函数性质）
- 解是唯一的（最高到平坦区域）

### ML 中的凸 vs 非凸

| 问题 | 凸？ | 原因 |
|------|------|------|
| 线性回归 (MSE) | 是 | 损失关于权重是二次的 |
| 逻辑回归 | 是 | 对数损失关于权重是凸的 |
| SVM（合页损失） | 是 | 线性函数的最大值 |
| LASSO（L1 回归） | 是 | 凸函数之和是凸的 |
| 岭回归（L2） | 是 | 二次 + 二次 = 凸 |
| 神经网络（任何损失） | 否 | 非线性激活创建非凸地形 |
| k-means 聚类 | 否 | 离散赋值步骤 |
| 矩阵分解 | 否 | 未知量的乘积 |

带凸损失的线性模型是凸的。一旦你添加带非线性激活的隐藏层，凸性就被破坏了。

### Hessian 矩阵

函数 f: R^n -> R 的 Hessian H 是二阶偏导数的 n x n 矩阵。

```
H[i][j] = d^2 f / (dx_i dx_j)
```

对 f(x, y) = x^2 + 3xy + y^2：

```
df/dx = 2x + 3y       d^2f/dx^2 = 2      d^2f/dxdy = 3
df/dy = 3x + 2y       d^2f/dydx = 3      d^2f/dy^2 = 2

H = [ 2  3 ]
    [ 3  2 ]
```

Hessian 告诉你曲率：
- 所有特征值为正：函数在每个方向上都向上弯曲（在该点凸）
- 所有特征值为负：在每个方向上向下弯曲（凹，局部最大值）
- 正负混合：鞍点（某些方向向上弯，某些方向向下弯）
- 零特征值：在该方向平坦（退化）

对于凸性，Hessian 必须处处是半正定的（所有特征值 >= 0），不仅仅在单个点。

### 牛顿法 (Newton's Method)

梯度下降使用一阶信息（梯度）。牛顿法使用二阶信息（Hessian）。它在当前点拟合一个二次近似，直接跳到该二次函数的最小值。

```
更新规则：
  x_new = x - H^(-1) * gradient

对比梯度下降：
  x_new = x - lr * gradient
```

牛顿法用逆 Hessian 替代标量学习率。这基于局部曲率自动调整步长和方向。

优势：
- 在最小值附近具有二次收敛性（误差每步平方）
- 不需要调整学习率
- 尺度不变（无论你如何参数化问题，工作方式相同）

劣势：
- 计算 Hessian 需要 O(n^2) 内存，求逆需要 O(n^3)
- 对有 100 万权重的神经网络，那是 10^12 个元素和 10^18 次操作
- 对深度学习不实用

### 约束优化

无约束优化：在所有 x 上最小化 f(x)。
约束优化：在约束条件下最小化 f(x)。

现实问题有约束。你想最小化成本但预算有限，你想最小化误差但模型复杂度受限。

### 拉格朗日乘数法 (Lagrange Multipliers)

拉格朗日乘数法将约束问题转换为无约束问题。

问题：在 g(x) = 0 的约束下最小化 f(x)。

解法：引入新变量（拉格朗日乘数 lambda）并求解无约束问题：

```
L(x, lambda) = f(x) + lambda * g(x)
```

在解处，L 的梯度为零：

```
dL/dx = df/dx + lambda * dg/dx = 0
dL/dlambda = g(x) = 0
```

几何直觉：在约束最小值处，f 的梯度必须平行于约束 g 的梯度。如果它们不平行，你可以沿约束曲面移动并进一步减小 f。

示例：在 x + y = 1 约束下最小化 f(x,y) = x^2 + y^2。

```
L = x^2 + y^2 + lambda(x + y - 1)

dL/dx = 2x + lambda = 0  =>  x = -lambda/2
dL/dy = 2y + lambda = 0  =>  y = -lambda/2
dL/dlambda = x + y - 1 = 0

从前两个：x = y
代入：2x = 1，所以 x = y = 0.5，lambda = -1
```

直线 x + y = 1 上距离原点最近的点是 (0.5, 0.5)。

### KKT 条件

Karush-Kuhn-Tucker 条件将拉格朗日乘数法扩展到不等式约束。

问题：在 g_i(x) <= 0, i = 1, ..., m 约束下最小化 f(x)。

KKT 条件（最优性的必要条件）：

```
1. 平稳性：       df/dx + sum(lambda_i * dg_i/dx) = 0
2. 原始可行性：   g_i(x) <= 0  对所有 i
3. 对偶可行性：   lambda_i >= 0  对所有 i
4. 互补松弛性：   lambda_i * g_i(x) = 0  对所有 i
```

互补松弛性是关键洞察：要么约束是激活的（g_i = 0，解位于边界上），要么乘数为零（约束没有影响）。不影响解的约束有 lambda = 0。

KKT 条件是 SVM 的核心。支持向量是约束激活（lambda > 0）的数据点。所有其他数据点有 lambda = 0，不影响决策边界。

### 正则化作为约束优化

L1 和 L2 正则化不是任意的技巧，它们是伪装成无约束问题的约束优化问题。

**L2 正则化（Ridge）：**

```
minimize  Loss(w)  subject to  ||w||^2 <= t

等价的无约束形式：
minimize  Loss(w) + lambda * ||w||^2
```

约束 ||w||^2 <= t 定义一个球（2D 中是圆，3D 中是球）。解是损失等值线首次接触这个球的地方。

**L1 正则化（Lasso）：**

```
minimize  Loss(w)  subject to  ||w||_1 <= t

等价的无约束形式：
minimize  Loss(w) + lambda * ||w||_1
```

约束 ||w||_1 <= t 定义一个菱形（2D 中是旋转的正方形）。

| 性质 | L2 约束（圆） | L1 约束（菱形） |
|------|-------------|----------------|
| **约束形状** | 圆（高维是球） | 菱形（2D 中是旋转正方形） |
| **损失等值线接触位置** | 平滑边界——圆上任意点 | 角——与轴对齐 |
| **解的行为** | 权重小但非零 | 某些权重恰好为零（稀疏） |
| **结果** | 权重收缩 | 特征选择 |

这解释了为什么 L1 产生稀疏模型（特征选择）而 L2 只收缩权重。菱形有与轴对齐的角，损失等值线更可能在角处与约束相交，将一个或多个权重恰好设为零。

## 构建

### 步骤 1：用 Hessian 检验凸性

```python
import numpy as np

def is_convex_hessian(f_hessian, domain_samples):
    for x in domain_samples:
        H = f_hessian(x)
        eigenvalues = np.linalg.eigvalsh(H)
        if np.any(eigenvalues < -1e-10):
            return False
    return True
```

### 步骤 2：牛顿法

```python
def newton_method(f_grad, f_hessian, x0, tol=1e-8, max_iter=100):
    x = np.array(x0, dtype=float)
    for i in range(max_iter):
        grad = np.array(f_grad(x))
        hess = np.array(f_hessian(x))
        if np.linalg.norm(grad) < tol:
            break
        delta = np.linalg.solve(hess, -grad)
        x = x + delta
    return x
```

## 使用

将牛顿法应用于凸优化问题：

```python
import numpy as np

def rosenbrock(x):
    return (1 - x[0])**2 + 100 * (x[1] - x[0]**2)**2

def rosenbrock_grad(x):
    return np.array([
        -2*(1 - x[0]) - 400*x[0]*(x[1] - x[0]**2),
        200*(x[1] - x[0]**2)
    ])

def rosenbrock_hessian(x):
    return np.array([
        [2 - 400*x[1] + 1200*x[0]**2, -400*x[0]],
        [-400*x[0], 200]
    ])

x0 = np.array([-1.0, 1.0])
x_newton = newton_method(rosenbrock_grad, rosenbrock_hessian, x0)
print(f"Newton minimum: {x_newton}")
```

注意：Rosenbrock 函数在最小值附近是非常非凸的（一个狭长的弯曲谷），所以牛顿法可能发散，除非靠近解。使用阻尼牛顿法或信赖域法来实现鲁棒性。

在实践中，使用 PyTorch 的自动微分来计算 Hessian 乘积：

```python
import torch

def hessian_vector_product(loss, params, vector):
    grad = torch.autograd.grad(loss, params, create_graph=True)
    dot = sum((g * v).sum() for g, v in zip(grad, vector))
    hvp = torch.autograd.grad(dot, params)
    return hvp
```

## 关键术语

| 术语 | 人们说的 | 实际含义 |
|------|---------|---------|
| 凸集 | "没有凹痕的集合" | 集合中任意两点之间的线段也完全在集合内 |
| 凸函数 | "碗状函数" | 曲线始终位于弦下方的函数。每个局部最小值是全局最小值 |
| Hessian | "二阶导数矩阵" | 所有二阶偏导数的矩阵。描述函数的局部曲率 |
| 半正定 | "所有特征值 >= 0" | 对任何 v 有 v^T H v >= 0。在凸性中意味着函数向上弯曲 |
| 牛顿法 | "使用曲率信息的梯度下降" | 使用逆 Hessian 作为预条件子。二次收敛但每次迭代昂贵 |
| 拉格朗日乘数法 | "约束问题的微积分" | 通过添加变量将约束优化转化为无约束优化 |
| KKT 条件 | "带不等式的拉格朗日乘数法" | 约束最优性的必要条件。互补松弛性是关键 |
| 互补松弛性 | "约束要么激活，要么无关" | lambda_i * g_i(x) = 0。每个约束要么在边界上，要么忽略 |
| 对偶问题 | "从另一面求解" | 将原始问题转化为对偶问题，有时对偶问题更容易求解。SVM 的核心 |
| 二次规划 | "二次函数加线性约束" | 最小化 x^T Q x + c^T x 受 Ax <= b 约束。SVM 训练就是这个 |
| L1-L2 正则化几何 | "为什么 L1 产生零" | L1 = 菱形，L2 = 圆。损失等值线在尖角处碰到菱形（零权重） |
| 鞍点 | "最陡下降的平稳点" | Hessian 有正负特征值的点。在 DL 中比局部最小值更常见 |
