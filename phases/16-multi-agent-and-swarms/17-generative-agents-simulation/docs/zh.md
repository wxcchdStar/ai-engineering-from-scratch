# 生成式智能体（Generative Agents）与涌现式模拟（Emergent Simulation）

> Park 等人 2023（UIST '23, arXiv:2304.03442）用三部分架构填充了 **Smallville**，一个包含 25 个智能体的沙盒：**记忆流（memory stream）**（自然语言日志）、**反思（reflection）**（智能体对其自身流生成的高级综合）和**计划（plan）**（日级行为，然后是子计划）。里程碑式的结果是情人节派对的涌现：一个智能体被植入"想举办情人节派对"的种子，无需进一步脚本，就产生了在人群中传播的邀请、协调日期，派对发生了——来自 24 个最初对此一无所知的智能体。消融实验显示所有三个组件都是可信性（believability）所必需的。记录的故障是空间规范错误（进入关闭的商店、共享单人浴室）。这是 2026 年智能体模拟和多智能体社会评估的参考架构。

**类型：** 学习 + 构建
**语言：** Python（标准库）
**前置条件：** 第 16 阶段 · 04（原语模型）、第 16 阶段 · 13（共享内存）
**时间：** ~75 分钟

## 问题

大多数多智能体系统是紧密脚本化的团队：规划器规划，编码器编码，审查者审查。这对定义明确的任务有效。它不能捕捉当智能体拥有记忆、优先级和开放世界时出现的涌现式、非脚本化行为。研究、社会模拟以及越来越多的游戏 AI 需要这第二种。

Smallville 架构是其基准。在 Park 2023 之前，最好的智能体模拟是浅层脚本跟随者；之后，该模式是开放世界中生成式智能体的默认设置。如果你在 2026 年构建智能体模拟，你要么使用 Smallville 的三个组件，要么明确证明为什么不使用。

## 概念

### 三个组件

**记忆流（Memory stream）。** 观察、动作、反思和计划的仅追加日志。每个条目有时间戳、类型、描述（自然语言）和派生元数据：**新近度（recency）**、**重要性（importance）**（智能体自评 1-10）和**相关性（relevance）**（与当前查询的余弦相似度）。

```
[2026-02-14 09:12:03] 观察：Isabella Rodriguez 问我是否喜欢爵士乐
[2026-02-14 09:14:22] 反思：   我喜欢关于音乐的长时间对话
[2026-02-14 10:05:00] 计划：   今晚参加 Isabella 的情人节派对
```

记忆检索结合三个分数：`score = w_recency * e^(-decay * age) + w_importance * importance + w_relevance * cos_sim`。Top-k 条目进入当前提示词。

**反思（Reflection）。** 定期（每 N 条记忆或在重要事件时），智能体从最近的记忆中生成高级综合。反思条目回到流中，像任何其他记忆一样可检索。这是智能体如何构建"理解"——架构中等价于长期信念的东西。

**计划（Plan）。** 自上而下的分解。首先，粗略的日级计划（"上班，与 Klaus 共进晚餐"）。然后小时级计划。然后动作级计划。计划是可修正的：当观察与计划矛盾时，智能体重新规划受影响的部分。

### 为什么三者都重要（消融）

Park 等人运行了分别删除观察、反思和计划的消融实验。每次消融都会损害可信性：

-   没有**观察**，智能体错过上下文并根据过时的信念行动。
-   没有**反思**，智能体无法形成高级信念；交互保持浅层。
-   没有**计划**，行为变成反应性噪音；目标消散。

来自人类评分者的可信性分数在所有三个组件都存在时最高；删除任何一个都会产生可测量的退步。

### 情人节涌现

一个智能体，Isabella Rodriguez，被植入目标"想在 2 月 14 日下午 5 点在 Hobbs Cafe 举办情人节派对"。其他 24 个智能体没有收到这样的种子。在模拟的日子里：

1.  Isabella 的计划包括邀请人。
2.  每个邀请成为邻居记忆流中的观察。
3.  该邻居的反思生成信念："Isabella 正在举办派对。"
4.  邻居的计划包含"2 月 14 日参加派对"。
5.  邻居告诉其他邻居。邀请在没有中央协调的情况下传播。
6.  2 月 14 日下午 5 点，几个智能体聚集在 Hobbs Cafe。

这是技术意义上的涌现：系统级行为（派对）从局部交互（双边邀请 + 个人规划）中产生，没有中央编排器。

### 记录的故障模式

Park 等人明确记录了：

-   **空间规范错误。** 智能体走进关闭的商店。智能体试图使用同一个单人浴室。智能体在不适合进食的房间进食。模型不会仅从环境中推断社会物理规范。
-   **记忆溢出。** 深度模拟运行导致记忆检索成本增长。实际补救措施：定期记忆压缩（摘要和修剪）和低重要性条目的衰减。
-   **反思幻觉。** 反思可以发明记忆流中不存在的关系。缓解措施：在反思提示词中包含源记忆 ID，并在检索时验证。

这些是生产相关的故障模式：任何 2026 年智能体模拟都会继承它们。

### 三组件实现规则

1.  **记忆是仅追加的。** 永远不要修改记忆条目。更正是新条目。
2.  **重要性分数是廉价的。** 在写入时调用 LLM 评分重要性 1-10。缓存分数。
3.  **检索是排名的，不是过滤的。** 按组合分数 Top-k；不要使用硬过滤器（会丢失上下文）。
4.  **反思定期运行。** 当未处理记忆的重要性之和超过阈值（例如 150）时触发。
5.  **计划是可修正的。** 当新观察与计划矛盾时，仅重新生成受影响的部分，而不是整个计划。

### Smallville 之外的生成式智能体

2024-2026 年的后续文献扩展了该架构：

-   **用于政策/市场研究的多智能体社会模拟。** 类似 Smallville 的人群模拟用户对功能的响应行为。比 A/B 测试更快；准确性有争议。
-   **游戏的 NPC AI。** 带有 Smallville 智能体的 RPG 产生涌现式故事情节，而不是脚本化任务。
-   **生成式智能体评估基准。** 指标变为长时间运行中行为的可信性 + 连贯性，而不是任务准确性。

该架构是参考。扩展替换组件（用于记忆的向量存储、检索增强反思、神经符号计划）但保留三部分结构。

### 为什么这对多智能体工程很重要

Smallville 是概念证明，表明当组件正确时，多智能体涌现是廉价的。该架构现已在开源模型上复现（较小的 LLM 优雅地而非急剧地失去可信性）。任何需要**涌现式社会行为**的生产系统都使用这种形状。任何需要**紧密任务执行**的系统都使用本阶段早期的监督者/角色/原语模式。

## 构建它

`code/main.py` 用标准库 Python 实现三个组件，使用脚本化智能体策略（无真实 LLM）。演示以微型方式复现情人节派对涌现：

-   `MemoryStream`——带有新近度/重要性/相关性检索的仅追加日志。
-   `reflect(stream)`——对最近高重要性记忆的脚本化反思。
-   `plan(agent_state)`——基于当前信念的日级和小时级计划。
-   场景：5 个智能体。智能体 1 以"下午 5 点举办派对"开始。在模拟的时间步中，邀请传播，智能体聚集。

运行：

```
python3 code/main.py
```

预期输出：逐时间步跟踪。到最后的时间步，5 个智能体中至少有 3 个在其计划中显示派对，并且它们聚集在派对地点。单个种子在没有编排器的情况下产生了协调到达。

## 使用它

`outputs/skill-simulation-designer.md` 设计生成式智能体模拟：智能体数量、记忆模式、反思节奏、计划范围、评估指标。

## 交付它

生产模拟规则：

-   **记忆是数据库。** 大规模选择真正的存储（向量数据库、Postgres）。内存标准库用于原型。
-   **记录检索跟踪。** 对于每个动作，记录驱动它的 top-k 记忆。这是你的调试能力。
-   **每个智能体的 token 预算。** 每个智能体每个时间步的检索 + 反思 + 计划是 O(k) 次 LLM 调用。N 个智能体 × T 个时间步 × 每次调用的调用次数可能使你的预算相形见绌。
-   **定期压缩记忆。** 摘要和修剪低重要性条目。保留策略是一个设计决策，不是细节。
-   **显式检测空间/社会规范违规。** 架构不会学习它们。

## 练习

1.  运行 `code/main.py`。确认 3+ 个智能体在派对上聚集。将智能体增加到 10——涌现仍然发生吗？
2.  删除反思步骤。行为看起来像什么？映射到 Park 2023 的消融发现。
3.  引入一个竞争的种子目标（"Klaus 想在下午 5 点做研究报告"）。智能体会分裂，还是一个目标占主导？什么决定它？
4.  添加空间约束：Hobbs Cafe 最多容纳 4 个智能体。模拟是优雅地处理溢出，还是遇到"单人浴室"故障模式？
5.  阅读 Park 等人（arXiv:2304.03442）第 6 节（涌现行为实验）。识别一个在你的微型中不可复现的行为。你需要增强架构的哪个组件？

## 关键术语

| 术语 | 人们的说法 | 实际含义 |
|------|-----------|----------|
| 记忆流 | "智能体的日记" | 观察、动作、反思、计划的仅追加日志。 |
| 新近度 | "记忆有多新" | 按年龄的指数衰减分数。 |
| 重要性 | "智能体有多在乎" | 写入时自评 1-10。缓存。 |
| 相关性 | "与当前查询的相关程度" | 余弦相似度（基于嵌入）。 |
| 反思 | "高级信念" | 从最近记忆生成的综合，作为新记忆重新摄入。 |
| 计划 | "日/小时/动作分解" | 自上而下的计划树。当观察矛盾时可修正。 |
| Smallville | "Park 2023 的沙盒" | 产生情人节涌现的 25 智能体模拟。 |
| 可信性 | "质量指标" | 人类评分员对行为是否像合理智能体的评分。 |
